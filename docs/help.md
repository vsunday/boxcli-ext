================

* [`box-u-ext help [COMMAND]`](#box-u-ext-help-command)

## `box-u-ext help [COMMAND]`

display help for box-u-ext

```
USAGE
  $ box-u-ext help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
