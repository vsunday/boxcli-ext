`box-u-ext autocomplete`

* [`box-u-ext autocomplete [SHELL]`](#box-u-ext-autocomplete-shell)

## `box-u-ext autocomplete [SHELL]`

display autocomplete installation instructions

```
USAGE
  $ box-u-ext autocomplete [SHELL]

ARGUMENTS
  SHELL  shell type

OPTIONS
  -r, --refresh-cache  Refresh cache (ignores displaying instructions)

EXAMPLES
  $ box-u-ext autocomplete
  $ box-u-ext autocomplete bash
  $ box-u-ext autocomplete zsh
  $ box-u-ext autocomplete --refresh-cache
```
